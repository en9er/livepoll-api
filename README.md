# Livepoll-api

## Getting Started

1. **Clone the repository:**

    ```bash
    git clone https://github.com/en9er/livepoll-api.git
    cd livepoll-api
    ```
2. **Install poetry:**
   ```bash
    pip install poetry
    ``` 
3. **Install dependencies:**
   ```bash
    poetry install
    ```
4. **Run the server:**
    ```bash
    uvicorn main:app --reload
    ```
